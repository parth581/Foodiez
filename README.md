## ‚öôÔ∏è Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/parth581/Foodiez.git
    ```

2. Add Firebase config:
    - Download `google-services.json` from the Firebase Console  
    - Place it in `android/app/` directory

3. Run the project:
    ```bash
    flutter pub get
    flutter run
    ```
